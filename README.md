# Carolyne Liang — Artist Portfolio

Static portfolio website for composer, sound designer, and multichannel artist Carolyne (Chuowen) Liang.

## Preview locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

The site is designed for GitHub Pages. Enable Pages in the repository settings and deploy from the root of the default branch, or add a custom domain after its DNS records are configured.
